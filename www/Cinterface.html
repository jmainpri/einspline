C interface

libbspline organizes spline routines into two groups: those which are specified on a uniform grid (or multidimensional mesh), and those on nonuniform grids. The uniform versions of the routines are documented below. The nonuniform versions will be translated from C++ shortly.

Uniform splines:

Spline creation

Each B-spline object type has an associated creation routine, to the user must pass the dimensions of the data, the boundary conditions to use, and a pointer to the mesh data.
